Multi-Minesweeper Help

How to play

Multi-Minesweeper is a variant of classic Minesweeper where each cell can contain more than one mine (up to 6). The number shown on an opened cell is the total count of mines in all eight surrounding cells.

Controls

  • Left-click — Open a cell (or chord if already opened)
  • Right-click — Cycle marker up: none → flag 1 → flag 2 → … → none
  • Shift + Right-click — Cycle marker down (negative flags, for negative-mines mode)
  • Middle-click — Chord: open all non-flagged neighbours when flags match the hint

Toolbar

  • Settings — Choose a preset difficulty or configure a custom game
  • Help — This page
  • Smiley — Click to start a new game
  • Timer — Elapsed time since your first click
  • Hint — Toggle hint mode: your next click reveals a 3×3 area (flags mines, opens safe cells)
  • Give Up — Reveal the entire board

Mine Breakdown

Below the board you'll see a breakdown showing how many mines of each group (×1, ×2, …) have been flagged versus the total.
